

일반적으로 /usr/src/apache/bin,
본인 컴퓨터의 경우 /usr/local/apache/bin 에서 확인할 수 있다.

1. ab
- 아파치 밴치마킹 도구

2. apachectl
- 아파치 웹 서버의 실행, 종료, 재시작을 위한 도구

3. htpasswd
- 유저 및 암호 생성

4. htdigest
- 사용자 인증 파일을 생성하고 업데이트 하는 도구

5. httpd
- 아파치 웹 서버의 데몬 프로그램이다

6. logresolve
- 아파치 웹 서버의 로그 파일에서 IP 주소를 호스트 네임으로 변환하는 도구

7. rotatelogs
- 아파치 웹 서버의 로그로테이트를 위한 도구
