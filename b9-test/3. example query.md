## 1. 조회

```
query {
  userPostsAndComments(userId: 1) {
    id
    title
    content
    comments {
      id
      content
    }
  }
}

```

## 2. 유저 생성

```
mutation {
  createUser(email: "example@example.com", password: "password123", name: "John Doe") {
    id
    email
    name
  }
}

```

## 3. 게시글 생성

```
mutation {
  createPost(title: "My First Post", content: "This is the content of my first post.", published: true, authorId: 1) {
    id
    title
    content
    published
    author {
      id
      name
    }
  }
}

```


## 4. 댓글 생성

```
mutation {
  createComment(content: "This is a comment.", postId: 1) {
    id
    content
    post {
      id
      title
    }
  }
}

```
