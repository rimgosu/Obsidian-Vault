## 요구사항
첫번째,
```
<li class="choice-li" th:if="${question.choices.choice1 != ''}" th:text="'① ' + ${question.choices.choice1}"></li>
...
<li class="choice-li" th:if="${question.choices.choice5 != ''}" th:text="'⑤ ' + ${question.choices.choice5}"></li>
```
다음을 클릭할 수 있게 바꿔주고, a태그의 색은 없어야해.

두번째, 
위의 li 태그를 클릭할 때마다 

```
<input type="hidden" th:value="${T(rimgosu.cbthub.domain.question.OX).X}" name="choseMultipleChoiceAnswer1"/>  
...
<input type="hidden" th:value="${T(rimgosu.cbthub.domain.question.OX).X}" name="choseMultipleChoiceAnswer2"/>  
```

의 값이 T(rimgosu.cbthub.domain.question.OX).X 라면 T(rimgosu.cbthub.domain.question.OX).O 로 바뀌고, T(rimgosu.cbthub.domain.question.OX).O라면 T(rimgosu.cbthub.domain.question.OX).X로 값이 바뀌어야해.

세번째,
클릭할 때마다 위의 li태그의 색이 핑크색으로 바뀌어야하고, 색이 이미 핑크색이라면 다시 원래대로 돌아와야해.

다음 요구사항에 맞춰 코드를 짜고, 전체 코드를 보여줄 수 있어?
